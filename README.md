# Number Guessing Game

This project is a very simple number guessing game designed to help me begin learning C++.

It is moreso a learning resource than a project, nonetheless, all features work as intended.

## Game Flow

When starting out, you are asked which level of difficulty you would like to set the game to.

1. Easy - Where you are given 10 guesses.
2. Medium - Where you are given 5 guesses.
3. Hard - Where you are given 3 guesses.

Once you select a level of difficulty, the game begins and after each subsequent guess you are told where you got it right, or whether the solution is higher or lower.

If you manage to get the answer correct then the game will exit, telling you that you got the answer with N guesses remaining.
